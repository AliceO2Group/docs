Detailed Description

Component fetches raw data input objects in DDL format and extracts tracklets. It also instantiates a RawReader in order to be used with some reconstruction.

More information and examples can be found here (relative to $ALICE_ROOT):

– HLT/BASE/AliHLTComponent.h/.cxx, HLT/BASE/AliHLTProcessor.h/.cxx Interface definition and description – HLT/SampleLib: example implementations of components
