Detailed Description

RawDataSpan holds ranges pointing to parts of other containers This class helps with restricting the view of digits/trackets/... to part of a timeframe, e.g. data from one event, detector, padrow or MCM. It also provides methods to help with iterating over the data The main data members are public, and can be accessed without setters/getters.

template<typename keyfunc >
template std::vector< RawDataSpan > RawDataSpan::iterateBy< MCM_ID > ( )

Return a vector with one data span for each MCM that has digits, tracklets or both IterateBy provides a more flexible interface, and should replace this method. Return a vector with data spans, split according to the keyfunc struct The keyfunc struct must have a method key to calculate a key for tracklets and digits that uniquely identifies the span this digit or tracklet will be part of. The key must be monotonically increasing for the digits and tracklets
