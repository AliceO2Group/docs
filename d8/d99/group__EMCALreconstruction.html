enum o2::emcal::reconstructionerrors::GainError_t
strong

Errors appearing when merging gain types.

Errors can appear when an expected gain type is missing, either because the HG is saturated and the LG is missing or because the LG is found for an ADC below HG/LG transition and the HG is missing.

Enumerator
LGNOHG 

LG found below HG/LG transition, HG missing.

HGNOLG 

HG saturated, LG missing.

UNKNOWN_ERROR 

Unknown error code.
