93 // Defines the 2D Legendre Polynominals with coefficients:
94 // w(u, v) = c_00 +
95 // c_10 * u + c_11 * v +
96 // c_20 * 0.5 * (3 * u * u - 1) + c_21 * u * v + c_22 * (3 * v * v - 1) +
98 // for u&v in [-1.0, 1.0]
