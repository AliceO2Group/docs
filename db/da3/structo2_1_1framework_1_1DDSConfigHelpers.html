Detailed Description

Helper to dump DDS configuration to run in a deployed manner. out is a stream where the configuration will be printed workflow is the workflow to dump metadata is the metadata to dump workflowSuffix a suffix to add to all the ids in the workflow specs is the internal representation of the dataflow topology which we want to dump. executions is the transient parameters for the afore mentioned specifications the full command being used
